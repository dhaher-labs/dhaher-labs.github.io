About

What is Dhaher Labs?

Dhaher Labs is a personal lab run by one person in Lhokseumawe, Aceh. We build LLM tools, quant research dashboards, and automated workflows. That is it. No board of directors, no funding rounds, no office.

We write code to solve problems we actually have: managing multiple LLM providers, screening markets, automating repetitive tasks. If it does not work for us, we do not ship it.

Everything is open source because closed-source side projects make no sense. If the code helps you, great. If not, that is fine too.

Portfolio

What We Care About

Build things that solve real problems for us first. If others find it useful, even better.

Make quant and trading research tools that a single person can actually run.

Automate the boring parts of development so we can focus on the interesting parts.

Keep everything open. No hidden code, no bait-and-switch licensing.
